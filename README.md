# 🌍 World Layoffs Data Analysis (2020–2024)

## 📌 Project Overview
This project analyzes global layoffs data from 2020 to 2024 using Python and Pandas. The dataset was cleaned, transformed, and explored to identify trends across companies, industries, countries, and years.

The goal of this project is to demonstrate practical data cleaning and exploratory data analysis (EDA) skills while uncovering meaningful insights from real-world data.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 🧹 Data Cleaning Process
The following data cleaning steps were performed:

- Removed duplicate records
- Handled missing values
- Standardized text columns
- Converted date columns to datetime format
- Fixed inconsistent values
- Prepared data for analysis

---

## 📊 Exploratory Data Analysis
The analysis focuses on:

- Top companies by total layoffs
- Top countries affected by layoffs
- Industry-wise layoff trends
- Year-wise layoff analysis
- Monthly layoff trends
- Companies with 100% workforce layoffs

---

## 🔍 Key Insights
Some questions explored in this project:

1. Which companies had the highest number of layoffs?
2. Which countries were most affected?
3. Which industries experienced the most layoffs?
4. How did layoffs change over time?
5. Which companies laid off their entire workforce?
